# “第三波书店”项目的要求：

## 初始时：
* 在BookShop文件夹进创建项目
* 项目名称为“Web”
* 项目类型为“ASP .NET MVC4 Web”项目
* ".NET Framework"的版本为4.0
* 项目模板为“基本”模板
* 项目引擎为“ASPX”引擎
